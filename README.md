# Student Grade Manager – C++

## Description
A console-based student grade management system. Add students, record grades, view individual performance, and see class statistics.

## Features
- Add, remove, search, and display students
- Add grades (0-100) to students
- Calculate individual averages, highest, lowest grades
- Class statistics and grade distribution
- File persistence (students.txt)

## How to Compile and Run
```bash
g++ -o grade_manager main.cpp Student.cpp GradeManager.cpp
./grade_manager
